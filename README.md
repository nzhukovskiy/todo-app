# Todo app
Приложение для управления задачами. Поддерживается авторизация, создание задач, удаление, редактирование, пометка как выполненного.

## Запуск
 - Клонировать репозиторий в локальный каталог;
 - Скопировать файл с переменными окружения:
```sh
cp .env.example .env
```
 - Запустить docker compose:
```sh
docker compose up
```
- Приложение будет доступно по адресу localhost:80
